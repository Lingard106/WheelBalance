# 二轮平衡循迹小车

## 📌 项目简介

本项目为竞赛用二轮平衡循迹小车，具备自平衡、循迹、自动路径行驶功能。  
**当前开发板**：STM32F103C8T6（调试及前期开发）  
**目标开发板**：STM32F407ZGT6（最终竞赛主控，规则强制要求）

### ✅ 当前已完成（在 STM32F103C8T6 上）

- [x] **MPU6050 传感器驱动**（I2C 通信，零偏自动校准）
- [x] **姿态解算算法**（Mahony 互补滤波 + 陀螺仪 Z 轴零偏 EKF）
- [x] **欧拉角输出**（Roll / Pitch / Yaw，单位：度）
- [x] **五路循迹模块驱动**（TCRT5000，中断方式，状态实时更新）
- [x] 基础 RTOS 任务框架（FreeRTOS + CMSIS-RTOS v1）

### 🔜 待开发

- [ ] 平衡控制算法（PID）
- [ ] 电机驱动与速度闭环
- [ ] 路径规划与自动导航
- [ ] 声光提示模块
- [ ] 一键启动与状态机
- [ ] 移植至 STM32F407ZGT6

## 🎯 竞赛功能要求与得分点

| 序号 | 要求 | 分值 | 备注 |
|------|------|------|------|
| 1 | 一键启动，不扶立即稳定站立 3 秒，前后移动 ≤15cm | 10 | 平衡 PID 调参关键 |
| 2 | A → B 自动行驶并停车，声光提示 | 20 | 直线循迹 + 定点停车 |
| 3 | A→B→半弧→C→D→半弧→A，每点声光提示，≤45s | 20 | 组合路径跟踪 |
| 4 | A→C→半弧→B→D→半弧→A，每点声光提示，≤60s | 20 | 路径切换与识别 |
| 5 | 按要求 4 路径自动行驶 4 圈，用时越短越好 | 20 | 高速稳定控制 |
| 6 | 使用组委会发放的 STM32F407ZGT6 作为唯一主控 | 10 | 硬件合规 |

## 🛠️ 硬件平台

### 调试平台（当前）

- **主控**：STM32F103C8T6（72MHz，64KB RAM，128KB Flash）
- **姿态传感器**：MPU6050 / MPU6500（六轴，I2C 接口）
- **循迹模块**：5 路 TCRT5000 红外对管
- **电机驱动**：TB6612 / L298N（待接入）
- **电源**：7.4V 锂电池 + 降压模块

### 目标平台（最终提交）

- **主控**：STM32F407ZGT6（168MHz，192KB RAM，1MB Flash）
- 其他外设兼容，代码可平滑移植。
